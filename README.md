## 🌐 Network Analysis Tools

### 🔢 Python Subnet Calculator
A professional CLI utility for network engineers and security analysts to calculate IPv4 network boundaries using CIDR notation.

**Key Features:**
* **Validation:** Uses Python's `ipaddress` module to verify IP integrity.
* **Network Mapping:** Calculates Network IDs, Broadcast Addresses, and Netmasks.
* **Audit Logic:** Identifies if an IP range is Public or Private (RFC 1918).
* **Host Counting:** Automatically calculates total usable host addresses.

**How to Run:**
```bash
python3 subnet_calc.py
